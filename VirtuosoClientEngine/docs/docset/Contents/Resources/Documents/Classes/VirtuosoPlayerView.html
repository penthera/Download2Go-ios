VirtuosoPlayerView Class Reference

Inherits from UIView
Conforms to VirtuosoPlayer
Declared in VirtuosoPlayerView.h

Overview

VirtuosoPlayerView can be used as a fully functional player view or as a starting point for your own custom user interface.

If you use VirtuosoPlayerView for playback, you do not need to manually call the VirtuosoLogger’s logPlaybackStart or logPlaybackStopped methods. VirtuosoPlayerView will automatically record these events for you.
