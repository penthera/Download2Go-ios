VirtuosoSecureClock Class Reference

Inherits from NSObject
Declared in VirtuosoSecureClock.h

Overview

This class is used internally to the SDK to insure that the user cannot violate expiry, licensing, and business logic by modifying the device system clock.

+ instance

The singleton instance access method

+ (nonnull VirtuosoSecureClock *)instance

Return Value

Returns the VirtuosoSecureClock object instance.

Discussion

You must never instantiate a local copy of the VirtuosoSecureClock object. This object is intended to be a static singleton accessed through the instance method only. Instantiating a local copy will throw an exception.
