Overview

VirtuosoPlayerViewController can be used as a fully functional player or as a starting point for your own custom user interface.

If you use VirtuosoPlayerViewController for playback, you do not need to manually call the VirtuosoLogger’s logPlaybackStart or logPlaybackStopped methods. VirtuosoPlayerViewController will automatically record these events for you.
