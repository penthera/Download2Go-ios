VirtuosoRenditionSelectionDelegate Protocol Reference

Conforms to NSObject
Declared in VirtuosoVideoRendition.h

Overview

If an asset is assigned a video rendition selection delegate, then the choice the delegate makes will be honored, rather than the choice made by internal processing. If the delegate returns nil, or is not assigned, SDK internal rules will be applied.

– selectRenditionFromAvailableRenditions:forAsset: required method

Called whenever the video rendition is to be selected.

- (VirtuosoVideoRendition *_Nullable)selectRenditionFromAvailableRenditions:(NSArray<VirtuosoVideoRendition*> *)renditions forAsset:(VirtuosoAsset *)asset

Parameters

renditions

Array of VirtuosoVideoRendition objects representing available video renditions
asset

VirtuosoAsset The asset being parsed and to which the provided renditions belong

Return Value

The video rendition selected for download, or nil to allow the SDK internal rules to be used
