VirtuosoLoggerDelegate Protocol Reference

Conforms to NSObject
Declared in VirtuosoLogger.h

Overview

For those who wish to use their own logging system or report metrics to non-Virtuoso locations/trackers. Virtuoso will send all log events to the registered delegate if the VirtuosoLogger would have otherwise processed them.

– virtuosoEventOccurred:

Reports that a tracked event was logged.

- (void)virtuosoEventOccurred:(nonnull VirtuosoBaseEvent *)event

Parameters

event

An object with base type VIrtuosoBaseEvent. The event itself may be any one of the classes derived from VirtuosoBaseEvent.

– virtuosoConsoleLogLine:atLevel:

Reports console logs

- (void)virtuosoConsoleLogLine:(nonnull NSString *)line atLevel:(kVL_LoggingLevel)level

Parameters

line

NSString containing the console log line
level

The severity level of the log item

Discussion

For non-debug builds and when verbose console option is off, Virtuoso won’t send debug output to the console. Use this method to access all console log output.
