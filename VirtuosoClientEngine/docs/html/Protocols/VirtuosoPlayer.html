VirtuosoPlayer Protocol Reference

Conforms to NSObject
Declared in VirtuosoPlayer.h

Overview

When calling the playUsingPlayer: method of an VirtuosoAsset, you must provide a player object that follows this protocol. Virtuoso will automatically handle all playback-related setup, e.g. create/configure a VirtuosoClientHTTPServer instance. It’s your responsibility to configure the player and present it in the view hierarchy.
