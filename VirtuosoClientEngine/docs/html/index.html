Cache & Carry (Virtuoso) SDK - iOS Reference

{{page/title}}

This library, internally named Virtuoso, is the iOS component of Penthera’s “Cache & Carry” toolkit. The toolkit is designed to download large objects (TV shows, movies, commercials, firmware updates, apps, user-generated content, ads, etc.) from a remote HTTP server to a broad range of mobile devices. The library offers functionality for queue management, subscription to episodic feeds, configuration, asset windowing, alerts, device/user/account management, event logging, and secure playout of downloaded objects. The library supports flat files, and it also implements a complete HLS, HSS, and MPEG-DASH stack in order to support the queuing, downloading, and management of complex segmented media objects.
