VirtuosoLicenseDelegateHelper Class Reference

Inherits from NSObject
Conforms to VirtuosoLicenseManagerDelegate
Declared in VirtuosoLicenseDefaultDelegate.h

Overview

This class can be used to provide a delegate for LicenseManager. Create an instanace, add license configuration objects for each asset you need to process and the class will automatically handle the delegate callbacks.
