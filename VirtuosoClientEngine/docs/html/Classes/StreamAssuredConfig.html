  targetBitrate

Target bitrate SAM should try and maintain.

@property (nonatomic, readonly) long long targetBitrate

Discussion

A value of - 1 will let the player the choose the the value automatically. 0 will select and lowest bitrate. Any other value SAM will pick the closest available bitrate.
